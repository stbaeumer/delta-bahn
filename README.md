# Delta Bahn (webxdc)

Web form for planning train journeys and sending travel itineraries to a chat as an `.ics` calendar file.

## Features

### Switchable languages: Deutsch (de), English (en). Default: Deutsch.

### Trip fields: departure station, arrival station, departure date/time, arrival date/time, train type, train number, coach, seat, and notes.

### Live preview of the generated trip message inside the form.

### Save trip to local storage for later retrieval.

### Delete all trip data with confirmation.

### Send trip as a chat message with attached `.ics` calendar file (named `reiseplan_von_bis.ics`).

### Click attached file to import trip to calendar
